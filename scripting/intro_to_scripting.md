# Scripting

## what is scripting?
- small, does the job
- purpose:
  - automate tasks
  - manipulate data
  - performs specific action that doesnt require complex of full app

## characteristics of scripting
- Automation a repetitive task e.g.:
  - file manipulation
  - system admin
  - batch processing
  - deployment pipelines
- Task- specific 
  - To solve a particular problem
  - shorter dev cycle
- Interactivity - will vary
- Usuaully run from CLI
- minimalist design
## Main diff between programming and scripting

| characteristic | Script                                       | Programming                                                |
|----------------|----------------------------------------------|------------------------------------------------------------|
| Complexity     | Simple                                       | more complex data structures and logic                     |
| Purpose        | To automate tasks that are simple or routine | much wider purpose, like large applications or video games |
 | Execution     | Usually by interpreter                       | either interpreter or needs to be compiled                 |
| development    | shorter                                      | longer and more in SDLC phases |
| learning curve | Easier | software design patterns and more involved to learn |
